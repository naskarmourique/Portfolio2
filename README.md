> **"Every code is a conquest, every project a campaign."**

Welcome to **The Chronicle**, an immersive, medieval-themed 3D portfolio designed to showcase the intersection of Data Science and Full-Stack Development. This isn't just a resume; it's a cinematic experience through the digital archives of a modern developer.

---

## 🏰 The Realms (Sections)

- **⚔️ The Warrior's Oath**: A parchment-style reveal of professional objectives and career philosophy.
- **🛡️ The Armory**: Interactive weapon racks showcasing technical skills across Programming, AI, and Web Dev.
- **🗺️ Campaigns & Battles**: A strategic war map visualizing professional experience and career growth.
- **🖼️ Great Works**: A stained-glass gallery featuring major full-stack and data science projects.
- **📜 Scrolls of Honor**: A collection of certifications and digital achievements.
- **🏛️ The Scholar's Path**: A chronicle of educational milestones and academic pursuits.
- **🐦 The Raven's Call**: An atmospheric contact section for professional inquiries.

---

## 🛠️ Forged With (Tech Stack)

### **Frontend Mastery**
- **React 18 & TypeScript**: For a robust, type-safe modular architecture.
- **Three.js & R3F**: Powering the cinematic 3D battlefield and interactive models.
- **Framer Motion**: Orchestrating scroll-linked animations and smooth UI transitions.
- **Tailwind CSS**: Custom utility-first styling with a mystical medieval color palette.

### **Tools of the Trade**
- **Vite**: High-performance build tool for rapid development.
- **Radix UI**: Accessible primitive components for a polished user interface.
- **Lucide React**: Themed iconography adapted for the medieval aesthetic.

## 🚀 Embarking on the Journey

### Prerequisites
- [Node.js](https://nodejs.org/) (Latest LTS)
- [pnpm](https://pnpm.io/) (Recommended)

### Local Setup
1. **Clone the Archives**
   ```bash
   git clone https://github.com/naskarmourique/Portfolio2.git
   cd Portfolio2
   ```

2. **Install Dependencies**
   ```bash
   pnpm install
   ```

3. **Ignite the Forge**
   ```bash
   pnpm dev
   ```
